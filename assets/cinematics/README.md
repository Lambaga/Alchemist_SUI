# Cinematics Folder

Hier werden Video-Dateien für Cinematics abgelegt.

## Intro-Video

Platziere die Intro-Datei hier als:
- **`intro.mp4`** (bevorzugt)

### Technische Anforderungen:
- **Format:** MP4 (H.264 codec empfohlen)
- **Auflösung:** Beliebig (wird automatisch auf Bildschirmgröße skaliert)
- **Audio:** Optional (wird automatisch abgespielt wenn vorhanden)

### Steuerung:
- **ESC** oder **Space** - Video überspringen
- Das Video wird automatisch vor dem Spielstart abgespielt wenn man "Neues Spiel" startet.

## Weitere Videos
Weitere Cinematics können im Code hinzugefügt werden. Siehe `src/ui/video_player.py`.
