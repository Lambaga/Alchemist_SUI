# Item-Icons für das Inventar

Lege hier PNG-Bilder für die Inventar-Items ab.

## Dateinamen (exakt so benennen):

| Dateiname | Item |
|-----------|------|
| `holzstab.png` | Holzstab |
| `stahlerz.png` | Stahlerz |
| `mondstein.png` | Mondstein |
| `kristall.png` | Kristall |
| `goldreif.png` | Goldreif |
| `wasserkristall.png` | Wasserkristall |
| `feueressenz.png` | Feueressenz |
| `erdkristall.png` | Erdkristall |

## Empfohlene Größe:
- **48x48 Pixel** oder **64x64 Pixel** (PNG mit Transparenz)
- Quadratisch für beste Darstellung

## Beispiel:
```
assets/ui/items/
├── holzstab.png
├── stahlerz.png
├── mondstein.png
├── kristall.png
├── goldreif.png
└── README.md
```

Das Spiel lädt automatisch die Bilder, wenn sie vorhanden sind.
Falls ein Bild fehlt, wird stattdessen ein farbiger Platzhalter angezeigt.
